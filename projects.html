Uni Meet

An Android application that allows users to connect with other students at their university. Users can match with other students based on hobbies, student organizations, and classes that they are taking. All this information is displayed on user profiles and stored in a Firebase real-time database.

Spotify Bot

A React web application that generates a random playlist for Spotify users based on a random genre including a shuffle feature. Uses Spotify OAuth (Open Authorization) protocol to allow users to connect their Spotify accounts.

Twitter Bot

A Python application that uses Tweepy, an open-source Python library for accessing the Twitter API. The bot is programmed to tweet the current date and time in set intervals.

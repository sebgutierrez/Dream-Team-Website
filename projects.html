Spotify Bot

A react web application that generates a random playlist for Spotify users based on a random genre with shuffle feature. Uses Spotify OAuth (Open Authorization) protocol to allow users to connect their Spotify accounts.

Uni Meet

An Android application that allows users to connect with other students at their university. Users can match with other students based on hobbies, student orgs, and classes that they are taking. All this information is displayed on user profiles and stored in a Firebase real-time database.
